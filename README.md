# BankSecure IT Audit & Access Control Simulation

## Project Overview
This project simulates an IT audit and cybersecurity risk assessment for a fictional banking environment. It was designed to reflect real-world job responsibilities in IT audit, cybersecurity operations, risk management, and compliance.

The lab demonstrates how access controls, system configurations, monitoring, and audit processes are reviewed in a regulated financial environment.

---

## Objectives
- Simulate user onboarding and offboarding  
- Review system controls and operating procedures  
- Perform IT risk assessments  
- Monitor logs for fraud, misuse, and security events  
- Test compliance with security policies  
- Document findings and remediation recommendations

## Key Activities
- Created role-based user access across systems  
- Conducted access reviews and removed orphan accounts  
- Performed infrastructure risk assessment using NIST/ISO models  
- Reviewed backup, DR, and security policies  
- Simulated audit findings and compliance gaps  
- Generated audit & risk reports with recommendations
