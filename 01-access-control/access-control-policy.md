## 1. Purpose
To ensure that only authorized users can access BankSecure systems and data.

## 2. Scope
Applies to all employees, contractors, and third-party users.

## 3. Roles
| Role | Description |
|------|-------------|
| Teller | Performs customer transactions |
| Operations | Oversees daily processes |
| IT Admin | Manages systems & access |
| Auditor | Reviews system activities |
| Manager | Approves transactions & reports |

## 4. Access Controls
- Role-based access control (RBAC)
- Unique user IDs
- Strong password policy
- Multi-factor authentication (MFA)
- Quarterly access reviews
- Immediate deprovisioning upon exit

## 5. Logging & Monitoring
All access events must be logged and reviewed by IT Security.
