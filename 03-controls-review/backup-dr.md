# Backup \& Disaster Recovery Assessment

## Objective

Ensure availability and data protection.

## Risks

* Data loss
* Regulatory breach
* Extended downtime

## Controls

* Daily encrypted backups
* Offsite storage
* Annual DR test

|**SYSTEM**|**BACKUP FREQUENCY**|**DR TEST**|**STATUS**|
|-|-|-|-|
|Core banking|Daily|Yes|OK|
|Email Server|Weekly|No|Gap|
